# Assets Directory

This directory should contain:

## Logo
- Add your JC Encadernados logo image here
- Recommended format: PNG or SVG
- Suggested name: `logo.png` or `logo.svg`

## Book Images
- Add book cover images here
- These images will be used for book displays throughout the application
- Recommended format: JPG or PNG
- Use descriptive filenames for easy reference

## Usage
Import images in your components like:
```typescript
import logo from '@/assets/logo.png';
```

Or reference via URL in the database/forms using relative paths.
